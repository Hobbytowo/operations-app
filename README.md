# Mathematical operations app

## Description
Application to help children with dynamically generated basic mathematical operations (+, -, *, /).

## Live preview
[Link](https://basic-math-operations.netlify.com/)

## Built with
- [Electron](https://electronjs.org/)
- Vue.js
